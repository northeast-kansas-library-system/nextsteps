How to copy a bibliographic record using ShareIt

  1. Log in to ShareIt

    (if you don’t know how to log into ShareIt, please call NEKLS to ask for help)

  2. Search for a title by ISBN / UPC / Title / Author as you would for any other title

  3. Once you get a list of possible matches and find a record that matches, select the title you think matches your item best

  4. Once the title record is open, click on the “Download record” button to download the MARC record for this title

  5. This will download the records as a file onto your computer - usually into your “Downloads” folder

    (The downloaded file usually has a “.mrc” or “.dat” filename extension)

  6. To import the record into Next Search Catalog see the stage and import records instructions
