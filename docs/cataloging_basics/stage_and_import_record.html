Stage and import a Marc record

Whether you’ve downloaded a Marc bibliographic record from ShareIt or OCLC or Baker and Taylor or another source, you’ll need to follow these steps to import the record into Next Search Catalog.

  1. Open the “Cataloging” page in Next Search Catalog

  2. Click on “Stage MARC records for import”

  3. Click on “Browse” and go to the folder on your local computer where you saved the Record

    (this is usually the “Downloads” folder on Microsoft Windows computers)

  4. Select the file to import and click on “Upload file”

  5. Change the drop-down menus to match the hint instructions

  6. A progress bar will show you how the job is going

  7. Once the progress bar reaches 100%, click on “View batch” to finish the import

  8. If no matches are found, click on “Import this batch into the catalog”

    (If matches are found, you may be importing a duplicate record - please avoid importing duplicate records - call NEKLS and ask for help if necessary)

  9. After clicking import, an “enqued job” message will appear to tell you how the job is going

    System Message: WARNING/2 (source/include.rst, line 214)

    Cannot scale image! Could not get size from "images/cataloging_basics/stage_and_import_records_0090.*": Requires Python Imaging Library.

  10. After the records have been imported, you can view the imported records by clicking on the “Manage imported batch” link

  11. After clicking on “Manage imported batch” you’ll be taken to the “Manage staged marc” page. Clicking on the title of an imported record will take you to that record.

  12. Once bibliographic records have been added, see the instructions for adding items to records to continue

    (Please note that if a bibliograhic record does not have an item added to it within 14 days, the bibliographic record will automatically be deleted)
