Lists

What are “Lists” in Next Search Catalog

Lists give staff a place where they can create a list of bibliographic records from the catalog. Lists can be public or private and you can set lists so that anyone who can view the list can add or remove titles from the list.

Lists should not be confused with the “Cart” feature in Next Search Catalog. The “Cart” is a temporary place to store bibliographic records and is emptied every time you shut down your browser. Lists are permanent places to store bibliographic record lists on your account until you decide to delete the list.

List requirements and options

  • All lists must have a title

  • Lists can be sorted by:

    • Author

    • Title (default sort)

    • Copyright date (as defined in the 264$c of the Marc record)

    • Call number (as defined in the 050 or 082 of the Marc record)

    • Date added (to the list)

  • Lists can be public or private

    • Private (default - lists are only visible to the account that created them)

    • Public (Public lists can be seen by anyone using the OPAC)

  • Allow changes

    • Nobody (not even the owner of the list can modify the list)

    • Owner ownly (default)

    • Anyone seeing the list (Only an option on public lists)

Finding the “Lists” module

  • The main lists menu can be found on the homepage in the staff interface:

  • The main lists menu can also be found by selecting “Lists” from the “More” menu:

  • The lists menu gives you the options to create new lists, view “Your lists,” or view “Public lists”

Creating new lists

There are three ways to create a new list in the staff interface:

  1. You can create a list by clicking on “+New list” anywhere in the Lists module

  1. You can create a list from any set of search results

Adding and deleting items on a list

There are three ways to add items to a list:

  1. You can add titles to an existing list using item barcode numbers or bibliographic record id numbers

  1. You can add titles to a list from any set of search results

  1. You can add titles to a list from the details page of a bibliographic record

  1. To remove titles from a list, check the checkboxes next to the titles, then click on “Remove selected”

Managing and deleting lists

Managing lists

You can manage your existing lists through the lists module.

  1. To edit the name of an existing list that you’ve created, slect “Edit” from the lists menu

  1. From here you can modify the name of the list, change the way the list is sorted, make the list public or private, and control who is allowed to add titles to the list

Deleting lists

To delete a list that you’ve created select “Delete” from the lists menu. Please note that you can only delete your own lists
