Adding and deleting items on a list

There are three ways to add items to a list:

  1. You can add titles to an existing list using item barcode numbers or bibliographic record id numbers

  1. You can add titles to a list from any set of search results
  1. You can add titles to a list from the details page of a bibliographic record
  1. To remove titles from a list, check the checkboxes next to the titles, then click on “Remove selected”
