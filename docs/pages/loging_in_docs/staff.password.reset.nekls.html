Reset password by calling NEKLS

Because staff login accounts have a “SYSTEM LOGIN ONLY” account category, currently only system administrators and library directors can change the passwords on these accounts through the staff client. Since the only system administrators are at the NEKLS office, NEKLS staff can set a new password for an account.
