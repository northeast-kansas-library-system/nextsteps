Printing options at check out

  • There are 3 print options for check out
    1. “Full page summary” prints an 8 1/2 x 11 summary of current checkouts and account information
    2. “Receipt-all checkouts” prints a receipt sized list of all items the borrower currently has checked out
    3. “Receipt-today’s checkouts” prints a receipted list of the items the patron has borrowed today
