Adding and deleting items on a list
===================================

There are three ways to add items to a list:

1. You can add titles to an existing list using item barcode numbers or bibliographic record id numbers

  |lists_adding_deleting_0010.png|

  |lists_adding_deleting_0015.png|

2. You can add titles to a list from any set of search results

  |lists_adding_deleting_0020.png|

3. You can add titles to a list from the details page of a bibliographic record

  |lists_adding_deleting_0030.png|

4. To remove titles from a list, check the checkboxes next to the titles, then click on "Remove selected"

  |lists_adding_deleting_0040.png|
