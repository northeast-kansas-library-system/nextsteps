Report 001152 contents // Top 50 Bibs with Holds

Report technical information: 1152

ID number: 1152

Name: Top 50 Bibs with Holds

Group: Popular Reports

Created on: 2011-07-29 16:18:30

Modified on: 2021-04-01 15:05:07

This file generated: 2023-03-24 16:49:10

Notes:

Generates a list of the 50 bibliographic records with the greatest numbers of requests

  • Shows current data
  • Shows bibliographic records from all Next Search Cataloglibraries
  • grouped by biblionumber
  • sorted by request count (biggest first)
  • contains links to the bibliographic record

Notes:

This report is being replaced by report 2735 which shows all items with more than 20 requests.

Report created by HB.

SQL: 1152

 1  SELECT
 2    count(*) AS NUMBER_OF_REQUESTS,
 3    Concat(
 4      biblio.title,
 5      " ",
 6      ExtractValue(biblio_metadata.metadata, '//datafield[@tag="245"]/subfield[@code>="b"]')
 7    ) AS TITLE,
 8    Concat(
 9      '<a href=\"/cgi-bin/koha/catalogue/detail.pl?biblionumber=',
10      biblio.biblionumber,
11      '\" target="_blank">LINK</a>'
12    ) AS LINK_TO_BIBLIO,
13    Concat(
14      '<a href=\"/cgi-bin/koha/catalogue/search.pl?idx=kw&q=',
15      biblio.title,
16      '&op=and&idx=kw&op=and&idx=kw&limit=mc-itype%2Cphr%3ALOCALHOLD1&limit=mc-itype%2Cphr%3AMEDIA&limit=mc-itype%2Cphr%3ANEWMEDIA&limit=mc-itype%2Cphr%3AWALKIN1&sort_by=relevance" target="_blank">video</a>'
17    ) AS VIDEO_SEARCH,
18    Concat(
19      "<a href='https://www.youtube.com/results?search_query=official+trailer+",
20      REPLACE(SUBSTRING_INDEX(biblio.title,
21      "(", 1),
22      " ", "+"),
23      "' target='_blank'>Youtube</a>"
24    ) AS YOUTUBE
25  FROM
26    reserves
27    JOIN biblio ON reserves.biblionumber = biblio.biblionumber
28    JOIN biblio_metadata ON biblio_metadata.biblionumber = biblio.biblionumber
29  GROUP BY
30    biblio.biblionumber
31  ORDER BY
32    count(*) DESC
33  LIMIT 50
