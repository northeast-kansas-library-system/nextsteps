Notes:

Monthly overview - collection statistics

  • Shows item and holding counts for the previous calendar month
  • At all Next Search Catalog libraries
  • grouped and sorted by branch name

Notes:

Generates data for:

  • START_I = number of items owned on the first of the month
  • END_I = number of items owned on the last day of the month
  • ITEMS_ADD_L_M = items added last month
  • ITEMS_DEL_L_M = items deleted last month
  • TOTAL_HOLDINGS = number of bibliographic records where at least 1 item was held at this library on the first of the month

This report and these notes updated on 2022.03.10

SQL: 3418

 1  SELECT
 2    branches.branchname AS BRANCH,
 3    Coalesce(Sum(DISTINCT START_ITEMS.count), 0) AS START_I,
 4    Coalesce(Sum(DISTINCT END_ITEMS.count), 0) AS END_I,
 5    Coalesce(Sum(DISTINCT ITEMS_ADDED.count), 0) AS ITEMS_ADD_L_M,
 6    Coalesce(ITEMS_DELETED.count, 0) AS ITEMS_DEL_L_M,
 7    Coalesce(Sum(DISTINCT TOTAL_HOLDINGS.count), 0) AS TOTAL_HOLDINGS
 8  FROM branches
 9    LEFT JOIN (
10      SELECT
11        items.homebranch,
12        Count(*) AS count
13      FROM items
14      WHERE items.dateaccessioned < AddDate(Last_Day(SubDate(Now(), INTERVAL 2 MONTH)), 1)
15      GROUP BY items.homebranch
16      UNION
17      SELECT
18        deleteditems.homebranch,
19        Count(*) AS count
20      FROM deleteditems
21      WHERE deleteditems.dateaccessioned < AddDate(Last_Day(SubDate(Now(), INTERVAL 2 MONTH)), 1)
22        AND deleteditems.timestamp >= AddDate(Last_Day(SubDate(Now(), INTERVAL 2 MONTH)), 1)
23      GROUP BY deleteditems.homebranch
24    ) START_ITEMS ON branches.branchcode = START_ITEMS.homebranch
25    LEFT JOIN (
26      SELECT
27        items.homebranch,
28        Count(*) AS count
29      FROM items
30      WHERE items.dateaccessioned < AddDate(Last_Day(SubDate(Now(), INTERVAL 1 MONTH)), 1)
31      GROUP BY items.homebranch
32      UNION
33      SELECT
34        deleteditems.homebranch,
35        Count(*) AS count
36      FROM deleteditems
37      WHERE deleteditems.dateaccessioned < AddDate(Last_Day(SubDate(Now(), INTERVAL 1 MONTH)), 1)
38        AND deleteditems.timestamp >= AddDate(Last_Day(SubDate(Now(), INTERVAL 1 MONTH)), 1)
39      GROUP BY deleteditems.homebranch
40    ) END_ITEMS ON branches.branchcode = END_ITEMS.homebranch
41    LEFT JOIN (
42      SELECT
43        items.homebranch,
44        Count(*) AS count
45      FROM items
46      WHERE Month(items.dateaccessioned) = Month(Now() - INTERVAL 1 MONTH)
47        AND Year(items.dateaccessioned) = Year(Now() - INTERVAL 1 MONTH)
48      GROUP BY items.homebranch
49      UNION
50      SELECT
51        deleteditems.homebranch,
52        Count(*) AS count
53      FROM deleteditems
54      WHERE Month(deleteditems.dateaccessioned) = Month(Now() - INTERVAL 1 MONTH)
55        AND Year(deleteditems.dateaccessioned) = Year(Now() - INTERVAL 1 MONTH)
56      GROUP BY deleteditems.homebranch
57    ) ITEMS_ADDED ON branches.branchcode = ITEMS_ADDED.homebranch
58    LEFT JOIN (
59      SELECT
60        deleteditems.homebranch,
61        Count(*) AS count
62      FROM deleteditems
63      WHERE Month(deleteditems.timestamp) = Month(Now() - INTERVAL 1 MONTH)
64        AND Year(deleteditems.timestamp) = Year(Now() - INTERVAL 1 MONTH)
65      GROUP BY deleteditems.homebranch
66    ) ITEMS_DELETED ON branches.branchcode = ITEMS_DELETED.homebranch
67    LEFT JOIN (
68      SELECT items.homebranch,
69        Count(DISTINCT items.biblionumber) AS count
70      FROM items
71      WHERE items.dateaccessioned < AddDate(Last_Day(SubDate(Now(), INTERVAL 1 MONTH)), 1)
72      GROUP BY items.homebranch
73      UNION
74      SELECT
75        deleteditems.homebranch,
76        Count(DISTINCT deleteditems.biblionumber) AS count
77      FROM deleteditems
78      WHERE deleteditems.dateaccessioned < AddDate(Last_Day(SubDate(Now(), INTERVAL 1 MONTH)), 1)
79        AND deleteditems.timestamp >= AddDate(Last_Day(SubDate(Now(), INTERVAL 1 MONTH)), 1)
80      GROUP BY deleteditems.homebranch
81    ) TOTAL_HOLDINGS ON branches.branchcode = TOTAL_HOLDINGS.homebranch
82  GROUP BY branches.branchname
83  ORDER BY BRANCH
