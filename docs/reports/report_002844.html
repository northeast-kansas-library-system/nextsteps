Report 002844 contents // GHW - Deleted borrowers count - date range

Report technical information: 2844

ID number: 2844

Name: GHW - Deleted borrowers count - date range

Group: Borrowers

Sub-group: Patron Statistics

Created on: 2016-12-02 14:56:53

Modified on: 2018-04-16 11:05:22

Created by: gwilliams

This file generated: 2023-03-24 16:49:10

Notes:

Count of patrons deleted during the specified date range
(between the beginning of the day on the specified "START DATE" to the end of the day on the specified "END DATE")

  • Looks at patrons who were manually deleted - does not include patrons that were deleted in batch processes
  • Allows you to specify a patron home branch and patron category if you wish
  • grouped by patron home branch and patron category - includes total count by home branch and a total for the entire table
  • sorted by patron home branch and patron category

Notes:

Report created at the request of Paola Free Library.

This report can only look back to the previous 60 days because we are purging data from the action logs that is more than 60 days old.

Click here to run in a new window

SQL: 2844

 1  SELECT
 2    deletedborrowers.branchcode AS PATRON_HOME_LIBRARY,
 3    deletedborrowers.categorycode AS PATRON_CATEGORY,
 4    Count(deletedborrowers.borrowernumber) AS COUNT_OF_PATRONS_DELETED
 5  FROM
 6    action_logs JOIN
 7    deletedborrowers
 8      ON deletedborrowers.borrowernumber = action_logs.object
 9  WHERE
10    action_logs.module = 'MEMBERS' AND
11    deletedborrowers.branchcode LIKE <<Choose patron home library|LBRANCH>> AND
12    deletedborrowers.categorycode LIKE <<Choose patron category|LBORROWERCAT>> AND
13    (action_logs.timestamp BETWEEN <<between the beginning of the day on "START DATE"|date>>  AND (<<and the end of the day on "END DATE"|date>>+ INTERVAL 1 DAY)) AND
14    action_logs.action LIKE 'DELET%'
15  GROUP BY
16    PATRON_HOME_LIBRARY, PATRON_CATEGORY
17    WITH ROLLUP
