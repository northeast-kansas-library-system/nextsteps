Name: GHW - Card Prefix Report - Part 1

Created on: 2016-09-13 18:01:22

Modified on: 2021-02-25 09:53:22

Created by: gwilliams

This file generated: 2023-03-24 16:49:10

Notes:

Attempts to discern which card number ranges are in use at a specified library

  • reports on current cards in NExpress
  • Shows card prefixes at a specified patron homebranch
    with a specified number of prefix digits
    and where there are more than a specified number of cards with that prefix
  • grouped and sorted by patron home library and card number prefix
  • click on links to see if other libraries are using this same prefix identified in the report (report 2773)

Notes:

Report 2773 is the companion to this report

Report created at the request of Robin Hastings.

Report created by George Williams.

Click here to run in a new window

SQL: 2771

 1  SELECT
 2    <<Number of digits in card prefix|YNUMBER>> AS CARD_PREFIX_LENGTH,
 3    borrowers.branchcode AS BORROWERS_HOME_BRANCH,
 4    Left(borrowers.cardnumber, <<Number of digits in card prefix|YNUMBER>>) AS LIBRAY_CARD_PREFIX,
 5    LENGTH(borrowers.cardnumber) AS CARD_NUMBER_LENGTH,
 6    Count(borrowers.borrowernumber) AS LIBRARY_CARD_COUNT,
 7    CONCAT( '<a href=\"/cgi-bin/koha/reports/guided_reports.pl?phase=Run+this+report&reports=2773&sql_params=', (Left(borrowers.cardnumber, <<Number of digits in card prefix|YNUMBER>>)) ,'&limit=10000 \" target="_blank">Link to report 2773</a>' ) AS LINK_TO_PREFIX_REPORT_2
 8  FROM
 9    borrowers
10  WHERE
11    borrowers.branchcode LIKE <<Patron home branch|LBRANCH>>
12  GROUP BY
13    borrowers.branchcode, CARD_NUMBER_LENGTH, LIBRAY_CARD_PREFIX
14  HAVING
15    LIBRARY_CARD_COUNT >= <<Minimum card number count|YNUMBER>>
16  ORDER BY
17    borrowers.branchcode,
18    LIBRAY_CARD_PREFIX
