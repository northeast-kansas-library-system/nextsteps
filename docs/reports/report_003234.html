Report 003234 contents // GHW - Patron count by extended attribute

Report technical information: 3234

ID number: 3234

Name: GHW - Patron count by extended attribute

Group: Borrowers

Sub-group: Patron attributes

Created on: 2019-07-24 13:40:33

Modified on: 2019-07-25 08:30:25

Cache expiry in seconds: 300

Created by: gwilliams

This file generated: 2023-03-24 16:49:10

Notes:

Generates a count of patrons with extended attributes at your library

  • Counts current patrons with attributes
  • at the library you specify
  • allows you to specify a minimum patron count for displaying an attribute count (i.e. only show attributes assigned to at least X patrons)
  • grouped and sorted by patron home branch, attribute category, and attribute
  • contains links to report 3235 which will list all of the patrons at your library with the attribute specified

Notes:

Patrons can have multiple extended attributes on their accounts. Similarly, patrons may not have any extended attributes on their accounts. A total count of patrons from all of the rows of this report will not indicate an accurate count of total patrons at your library.

SQL: 3234

 1  SELECT
 2    columncreator.branchcode AS LIBRARY,
 3    Concat_Ws(' // ', columncreator.CAT_DESC, columncreator.AV_DESC) AS EXTENDED_ATTRIBUTE,
 4    Count(borrowerss.borrowernumber) AS PATRON_COUNT,
 5    Concat('<a href="/cgi-bin/koha/reports/guided_reports.pl?reports=3235&phase=Run+this+report&param_name=Choose+your+library%7CLBRANCH&sql_params=', columncreator.branchcode, '&param_name=Choose+attribute+type%7CQ_BORROWER_ATTRIBUTES_A&sql_params=', columncreator.CAT_CODE, '&param_name=Enter+attribute+value+or+a+%25+symbol&sql_params=', IF(columncreator.AV_CODE = 'X', '%', columncreator.AV_CODE), ' " target="_blank">Go to detailed report</a>') AS LINK_TO_DETAILED_REPORT
 6  FROM
 7    (
 8      SELECT
 9        branches.branchcode,
10        attributess.CAT_DESC,
11        attributess.CAT_CODE,
12        attributess.AV_DESC,
13        attributess.AV_CODE
14      FROM
15        branches,
16        (
17          SELECT
18            borrower_attribute_types.code AS CAT_CODE,
19            borrower_attribute_types.description AS CAT_DESC,
20            If(authorised_values.authorised_value IS NULL, 'X', authorised_values.authorised_value) AS AV_CODE,
21            authorised_values.lib AS AV_DESC
22          FROM
23            borrower_attribute_types
24            LEFT JOIN authorised_values
25              ON authorised_values.category = borrower_attribute_types.authorised_value_category
26        ) attributess
27      ORDER BY
28        branches.branchcode
29    ) columncreator
30    LEFT JOIN (
31      SELECT
32        borrowers.branchcode,
33        borrower_attributes.code,
34        borrower_attributes.borrowernumber,
35        If(borrower_attribute_types.authorised_value_category = '', 'X', borrower_attributes.attribute) AS attribute,
36        borrower_attribute_types.authorised_value_category
37      FROM
38        borrowers
39        INNER JOIN borrower_attributes
40          ON borrower_attributes.borrowernumber = borrowers.borrowernumber
41        LEFT JOIN borrower_attribute_types
42          ON borrower_attributes.code = borrower_attribute_types.code
43      ORDER BY
44        borrowers.branchcode,
45        borrower_attributes.code,
46        borrower_attributes.attribute,
47        borrower_attributes.borrowernumber
48    ) borrowerss
49      ON borrowerss.branchcode = columncreator.branchcode AND
50        borrowerss.code = columncreator.CAT_CODE AND
51        borrowerss.attribute = columncreator.AV_CODE
52  WHERE
53    columncreator.branchcode LIKE <<Choose your library|ZBRAN>> AND
54    columncreator.CAT_CODE LIKE <<Extended attribute category|Q_BORROWER_ATTRIBUTES_A>>
55  GROUP BY
56    columncreator.branchcode,
57    columncreator.CAT_CODE,
58    columncreator.AV_CODE
59  HAVING
60    PATRON_COUNT >= <<View rows with at least X patrons|YNUMBER>>
61  ORDER BY
62    LIBRARY,
63    EXTENDED_ATTRIBUTE
