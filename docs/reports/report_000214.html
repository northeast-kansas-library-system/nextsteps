Name: GHW - Items with important fields that are blank or have problematic values

Created on: 2009-01-26 12:07:07

Modified on: 2023-03-24 16:47:18

Cache expiry in seconds: 500

This file generated: 2023-03-24 16:49:10

Notes:

Shows item records with important fields that are blank or may be problematic

  • Shows items currently in the catalog with the following fields left blank (or in some cases "Unclassified"):

    • barcode
    • permanent shelving location (blank or cataloging, processing, or recently returned)
    • item type (blank or unclassified)
    • collection code (blank or unclassified)
    • call number
    • replacement price

  • at the library you specify; and with the blank field you specify
  • grouped by biblio number and item number
  • sorted by item home branch, location, item type, collection code, call number, author, and title
  • contains links to the bibliographic record

Replaces reports:

  • 214 - Null Report - Missing Item Type report for all libraries (previous version)
  • 1285 - Null Report - Missing Collection Codes all
  • 1398 - Collection Codes to Fix
  • 1401 - Item Types to Fix

  • 1404 - Shelving Locations to fix

  • 1782 - Home or Current Branch is Null

  • 3362 - GHW - Items at a library without a replacement cost

  9      '<a class="btn btn-outline-primary noprint" ',
 10      'href="/cgi-bin/koha/catalogue/moredetail.pl?type=&itemnumber=',
 11      items.itemnumber,
 12      '&biblionumber=',
 13      biblio.biblionumber,
 14      '" target="_blank">Item record</a>'
 15    ) AS ITEM_LINK,
 16    biblio.biblionumber AS BIBLIO_NUMBER,
 17    items.itemnumber AS ITEM_NUMBER,
 18    items.barcode,
 19    items.homebranch,
 20    items.holdingbranch,
 21    plocs.lib AS "PERMANENT LOCATION",
 22    locs.lib AS LOCATION,
 23    itypes.description AS ITYPE,
 24    ccodes.lib AS CCODE,
 25    items.itemcallnumber AS "Call number",
 26    items.copynumber,
 27    biblio.author,
 28    Concat_Ws(" ", biblio.title, biblio.subtitle, biblioitems.number,
 29    biblio.part_name) AS "Full title",
 30    items.replacementprice AS "Replacement price",
 31    items.dateaccessioned AS "Date added",
 32    Concat_Ws("",
 33      If(items.barcode IS NULL, "Barcode is blank / ", ""),
 34      If(plocs.lib IS NULL, "Permanent location is blank / ",
 35        If(plocs.lib LIKE "%Cataloging%", "Permanent location = cataloging / ",
 36          If(plocs.lib LIKE "%Processing%", "Permanent location = processing / ",
 37            If(plocs.lib LIKE "%Recently%", "Permanent location = recently returned / ", "")
 38          )
 39        )
 40      ),
 41      If(ccodes.lib IS NULL, "Collection code is blank / ",
 42        If(ccodes.lib LIKE "%(UN%", "Collection code = (Unclassified) / ", "")
 43      ),
 44      If(itypes.description IS NULL, "Item type is blank ",
 45        If(itypes.description LIKE "%(UN%", "Item type = (Unclassified) / ", "")
 46      ),
 47      If(items.itemcallnumber IS NULL, "Call number is blank / ", ""),
 48      If(items.replacementprice IS NULL, "Replacement price is blank / ",
 49        If(items.replacementprice = 0, "Replacement price = $0.00 / ", "")
 50      )
 51    ) AS "Problem field"
 52  FROM
 53    items JOIN
 54    biblio ON items.biblionumber = biblio.biblionumber JOIN
 55    biblioitems ON biblioitems.biblionumber = biblio.biblionumber AND
 56        items.biblioitemnumber = biblioitems.biblioitemnumber LEFT JOIN
 57    (
 58      SELECT
 59        authorised_values.category,
 60        authorised_values.authorised_value,
 61        authorised_values.lib
 62      FROM
 63        authorised_values
 64      WHERE
 65        authorised_values.category = 'LOC'
 66    )
 67    plocs ON plocs.authorised_value = items.permanent_location LEFT JOIN
 68    (
 69      SELECT
 70        authorised_values.category,
 71        authorised_values.authorised_value,
 72        authorised_values.lib
 73      FROM
 74        authorised_values
 75      WHERE
 76        authorised_values.category = 'LOC'
 77    )
 78    locs ON locs.authorised_value = items.location LEFT JOIN
 79    (
 80      SELECT
 81        authorised_values.category,
 82        authorised_values.authorised_value,
 83        authorised_values.lib
 84      FROM
 85        authorised_values
 86      WHERE
 87        authorised_values.category = 'ccode'
 88    )
 89    ccodes ON ccodes.authorised_value = items.ccode LEFT JOIN
 90    (
 91      SELECT
 92        itemtypes.itemtype,
 93        itemtypes.description
 94      FROM
 95        itemtypes
 96    )
 97    itypes ON itypes.itemtype = items.itype
 98  WHERE
 99    items.homebranch LIKE <<Item home library|ZBRAN>> AND
100    ((items.barcode IS NULL) OR
101      (plocs.lib IS NULL) OR
102      (plocs.lib LIKE "%Cataloging%") OR
103      (plocs.lib LIKE "%Processing%") OR
104      (plocs.lib LIKE "%Recently%") OR
105      (ccodes.lib IS NULL) OR
106      (ccodes.lib LIKE "%(UN%") OR
107      (itypes.description IS NULL) OR
108      (itypes.description LIKE "%(UN%") OR
109      (items.itemcallnumber IS NULL) OR
110      (items.replacementprice IS NULL)) AND
111    Concat(
112      If(items.barcode IS NULL, "BE", ""),
113      If(plocs.lib IS NULL, "LE3",
114        If(plocs.lib LIKE "%Cataloging%", "LG3",
115          If(plocs.lib LIKE "%Processing%", "LG3",
116            If(plocs.lib LIKE "%Recently%", "LG3", "")
117          )
118        )
119      ),
120      If(ccodes.lib IS NULL, "CE3",
121        If(ccodes.lib LIKE "%(UN%", "CG3", "")
122      ),
123      If(itypes.description IS NULL, "IE3",
124        If(itypes.description LIKE "%(UN%", "IG3", "")
125      ),
126      If(items.itemcallnumber IS NULL, "NE", ""),
127      If(items.replacementprice IS NULL, "PE",
128        If(items.replacementprice = 0, "PE", "")
129      )
130    ) LIKE Concat("%", <<Empty or problematic field in item record|XS_NULLS>>, "%")
131  GROUP BY
132    biblio.biblionumber,
133    items.itemnumber
134  ORDER BY
135    items.homebranch,
136    PERMANENT_LOCATION,

138    CCODE,
139    items.itemcallnumber,
140    items.copynumber,
