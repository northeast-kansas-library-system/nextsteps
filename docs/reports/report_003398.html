Report 003398 contents // GHW - Borrower count by age

Report technical information: 3398

ID number: 3398

Name: GHW - Borrower count by age

Group: Borrowers

Sub-group: Patron attributes

Created on: 2021-01-11 16:32:52

Modified on: 2021-01-11 16:38:28

Cache expiry in seconds: 300

Created by: gwilliams

This file generated: 2023-03-24 16:49:10

Notes:

Counts borrowers at a library by age (in years) and gender indicator

  • Counts current borrowers
  • where the borrowers home library matches the library you specify
  • grouped and sorted by borrowers home branch and age

Notes:

Counts in the "BLANK_OR_BAD_CODE" column represent borrowers where the gender radio button on their account is either not set, or is set to a data code that is no longer in use.

Partly replaces report 1081.

SQL: 3398

 1  SELECT
 2    branches.branchname,
 3    ages.BORROWER_AGE AS BORROWER_AGE,
 4    Coalesce(ages.COUNT_OF_BORROWERS, '-') AS TOTAL_BORROWERS,
 5    Coalesce(FEMALE.COUNT_OF_BORROWERS, '-') AS FEMALE,
 6    Coalesce(MALE.COUNT_OF_BORROWERS, '-') AS MALE,
 7    Coalesce(NAS.COUNT_OF_BORROWERS, '-') AS NOT_SPECIFIED,
 8    Coalesce(BLANK_OR_UNKNOWN.COUNT_OF_BORROWERS, '-') AS BLANK_OR_BAD_CODE
 9  FROM
10    branches JOIN
11    (SELECT
12        borrowers.branchcode,
13        Coalesce(floor(DateDiff(CurDate(), borrowers.dateofbirth) / 365.25), '-') AS BORROWER_AGE,
14        Count(borrowers.borrowernumber) AS COUNT_OF_BORROWERS
15      FROM
16        borrowers
17      GROUP BY
18        borrowers.branchcode,
19        Coalesce(floor(DateDiff(CurDate(), borrowers.dateofbirth) / 365.25),
20        '-')) ages ON ages.branchcode = branches.branchcode LEFT JOIN
21    (SELECT
22        borrowers.branchcode,
23        Coalesce(floor(DateDiff(CurDate(), borrowers.dateofbirth) / 365.25), '-') AS BORROWER_AGE,
24        Count(borrowers.borrowernumber) AS COUNT_OF_BORROWERS
25      FROM
26        borrowers
27      WHERE
28        borrowers.sex LIKE 'N'
29      GROUP BY
30        borrowers.branchcode,
31        Coalesce(floor(DateDiff(CurDate(), borrowers.dateofbirth) / 365.25), '-')) NAS
32        ON NAS.branchcode = branches.branchcode AND
33        NAS.BORROWER_AGE = ages.BORROWER_AGE LEFT JOIN
34    (SELECT
35        borrowers.branchcode,
36        Coalesce(floor(DateDiff(CurDate(), borrowers.dateofbirth) / 365.25), '-') AS BORROWER_AGE,
37        Count(borrowers.borrowernumber) AS COUNT_OF_BORROWERS
38      FROM
39        borrowers
40      WHERE
41        borrowers.sex LIKE 'F'
42      GROUP BY
43        borrowers.branchcode,
44        Coalesce(floor(DateDiff(CurDate(), borrowers.dateofbirth) / 365.25), '-')) FEMALE
45        ON FEMALE.branchcode = branches.branchcode AND
46        FEMALE.BORROWER_AGE = ages.BORROWER_AGE LEFT JOIN
47    (SELECT
48        borrowers.branchcode,
49        Coalesce(floor(DateDiff(CurDate(), borrowers.dateofbirth) / 365.25), '-') AS BORROWER_AGE,
50        Count(borrowers.borrowernumber) AS COUNT_OF_BORROWERS
51      FROM
52        borrowers
53      WHERE
54        borrowers.sex LIKE 'M'
55      GROUP BY
56        borrowers.branchcode,
57        Coalesce(floor(DateDiff(CurDate(), borrowers.dateofbirth) / 365.25), '-')) MALE
58        ON MALE.branchcode = branches.branchcode AND
59        MALE.BORROWER_AGE = ages.BORROWER_AGE LEFT JOIN
60    (SELECT
61        borrowers.branchcode,
62        Coalesce(floor(DateDiff(CurDate(), borrowers.dateofbirth) / 365.25), '-') AS BORROWER_AGE,
63        Count(borrowers.borrowernumber) AS COUNT_OF_BORROWERS
64      FROM
65        borrowers
66      WHERE
67        (borrowers.sex IS NULL) OR
68        (borrowers.sex <> 'N' AND
69            borrowers.sex <> 'F' AND
70            borrowers.sex <> 'M')
71      GROUP BY
72        borrowers.branchcode,
73        Coalesce(floor(DateDiff(CurDate(), borrowers.dateofbirth) / 365.25), '-')) BLANK_OR_UNKNOWN
74        ON BLANK_OR_UNKNOWN.branchcode =
75        branches.branchcode AND
76        BLANK_OR_UNKNOWN.BORROWER_AGE = ages.BORROWER_AGE
77  WHERE
78    branches.branchcode LIKE <<Choose your library|ZBRAN>>
79  GROUP BY
80    branches.branchname,
81    ages.BORROWER_AGE
82  ORDER BY
83    branches.branchcode,
84    LPad(ages.BORROWER_AGE, 4, 0)
