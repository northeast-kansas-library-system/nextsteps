Report 002861 contents // GHW - Top 200 circulations

Report technical information: 2861

ID number: 2861

Name: GHW - Top 200 circulations

Group: Circulation

Sub-group: Circ Stats

Created on: 2016-12-19 10:09:30

Modified on: 2022-01-05 12:12:15

Created by: gwilliams

This file generated: 2023-03-24 16:49:10

Notes:

Shows the check-out count for the top 200 most circulated items

  • Shows check-outs during the specified date range
    (between the beginning of the day on the specified "START DATE" to the end of the day on the specified "END DATE")
  • Can be limited to items checked out at a specified library, with a specified home branch, shelving location, item type, and collection code
  • grouped by author and title
  • sorted by most check-outs to least check-outs
  • limited to 200 rows

Notes:

This report was updated and fixed on January 5, 2022

Please note that, because this report gathers data from the issues and oldissues tables in Koha, you cannot see accurate data from more than 13 months old.

Also note that this report cannot count the circulation of items that have been deleted. If an item circulates 50 times on Tuesday, but is deleted on Wednesday, it will not be counted by this report if the report is run on Thursday.

The report will almost certainly fail if you use extremely broad parameters. If you try to run this report with "All libraries" as the checkout branch, "All libraries" as the item home branch, and no limits on shelving locations, item types, or collection codes and a date range longer than just a few weeks, the report will almost certainly fail and may slow down Koha while it runs.

 1  SELECT
 2    Concat('<a href=\"/cgi-bin/koha/catalogue/detail.pl?biblionumber=',
 3    items.biblionumber, '\" target="_blank">', items.biblionumber, '</a>') AS
 4    LINK_TO_TITLE,
 5    biblio.author,
 6    Concat_Ws(' ',
 7      biblio.title,
 8      ExtractValue(biblio_metadata.metadata, '//datafield[@tag="245"]/subfield[@code="b"]'),
 9      ExtractValue(biblio_metadata.metadata, '//datafield[@tag="245"]/subfield[@code="p"]'),
10      ExtractValue(biblio_metadata.metadata, '//datafield[@tag="245"]/subfield[@code="n"]')
11    ) AS TITLE,
12    Count(DISTINCT allissues.issue_id) AS CIRCULATION_COUNT
13  FROM
14    (SELECT
15        issues.issue_id,
16        issues.itemnumber,
17        issues.branchcode,
18        issues.issuedate
19      FROM
20        issues
21      WHERE
22        issues.branchcode LIKE <<Choose check-out branch|ZBRAN>> AND
23        issues.issuedate BETWEEN
24          <<Count checkouts between the beginning of the day on|date>> AND
25          (<<and the end of the day on|date>> + INTERVAL 1 DAY)
26      GROUP BY
27        issues.issue_id,
28        issues.itemnumber,
29        issues.branchcode,
30        issues.issuedate
31      UNION
32      SELECT
33        old_issues.issue_id,
34        old_issues.itemnumber,
35        old_issues.branchcode,
36        old_issues.issuedate
37      FROM
38        old_issues
39      WHERE
40        old_issues.branchcode LIKE <<Choose check-out branch|ZBRAN>> AND
41        old_issues.issuedate BETWEEN
42          <<Count checkouts between the beginning of the day on|date>> AND
43          (<<and the end of the day on|date>> + INTERVAL 1 DAY)
44      GROUP BY
45        old_issues.issue_id,
46        old_issues.itemnumber,
47        old_issues.branchcode,
48        old_issues.issuedate) allissues  JOIN
49    items ON items.itemnumber = allissues.itemnumber  JOIN
50    biblio ON items.biblionumber = biblio.biblionumber  JOIN
51    biblio_metadata ON biblio_metadata.biblionumber = items.biblionumber
52  WHERE
53    items.homebranch LIKE <<Choose item homebranch|LBRANCH>> AND
54    items.permanent_location LIKE <<Choose item permanent shelving location|LLOC>> AND
55    items.itype LIKE <<Choose item type|LITYPES>> AND
56    items.ccode LIKE <<Choose item collection code|LCCODE>>
57  GROUP BY
58    biblio.biblionumber
59  ORDER BY
60    CIRCULATION_COUNT DESC
61  LIMIT 200
