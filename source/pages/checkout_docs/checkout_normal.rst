To check items out:
===================

1. Scan the borrower's library card number in the flexible search bar that's set to "Check out"

    - You can also type in a card number to look up a borrower's account
    - You can also type in a name to search for a borrower

|checkout.0050.png|

2. Scan an item barcode in the "Enter item barcode" input box

    - You can also type in an item barcode number

|checkout.0060.png|

3. The "Checked out" area will update with the title and barcode number of the item and the item's due date

|checkout.0070.png|
